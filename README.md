# 要睡了嘞 - 睡前陪伴小程序

这是一款睡前陪伴微信小程序，提供助眠故事、白噪音、树洞倾诉和互动投票等功能，帮助用户放松身心，安然入睡。

## 功能特性

### 核心功能
- **晚安故事**：精选睡前故事，伴你入眠
- **助眠声音**：多种白噪音（雨声、海浪、柴火、轻音乐）
- **失眠树洞**：倾诉心声，获得温暖回应
- **互动投票**：发起和参与投票，增强社交互动

### 社交互动功能
- 发起投票：创建投票主题，设置多个选项和截止时间
- 参与投票：查看投票列表，选择心仪选项
- 结果展示：实时显示投票进度和百分比
- 防重复投票：每人每个投票只能参与一次
- 时间限制：支持设置投票截止时间
- 数据上云：所有投票数据存储在云数据库，多人共享

## 技术栈

- **前端**：微信小程序原生开发
- **后端**：微信云开发
  - 云数据库：存储投票数据
  - 云函数：处理投票业务逻辑
  - 云存储：存储图片等资源

## 项目结构

```
├── cloudfunctions/          # 云函数目录
│   └── quickstartFunctions/ # 主云函数
│       └── index.js         # 包含投票相关云函数
└── miniprogram/             # 小程序前端
    ├── pages/               # 页面目录
    │   ├── index/           # 首页
    │   ├── story/           # 故事页面
    │   ├── sound/           # 声音页面
    │   ├── treehole/        # 树洞页面
    │   ├── vote/            # 投票页面
    │   │   ├── index.js     # 投票列表和发起投票
    │   │   └── detail.js    # 投票详情和参与投票
    │   └── mine/            # 个人中心
    └── app.js               # 应用入口
```

## 云数据库集合

- **votes**：投票数据集合
  - title: 投票标题
  - options: 选项数组（包含文本和票数）
  - deadline: 截止时间
  - voters: 已投票用户 openid 数组
  - createdAt: 创建时间
  - createdBy: 创建者 openid
  - status: 状态（active/ended）

## 快速开始

1. 克隆项目到本地
2. 使用微信开发者工具打开项目
3. 在云开发控制台创建 `votes` 集合
4. 部署云函数 `quickstartFunctions`
5. 编译运行小程序
